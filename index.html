História da Internet:

O surgimento da internet se deu com o desenvolvimento de pesquisas na área de comunicação e tecnologia. A guerra fria foi o agente principal que estimulou o surgimento da internet, pois a polarização mundial entre EUA X União Soviética levaram os Estados Unidos a investirem em pesquisas para elevar a proteção de suas informações

Sendo assim, a internet teve seu início justamente em pesquisas militares que levaram a criação de núcleos de pesquisas como a DARPA (Defense Advanced Research Project) e a ARPANET onde foi criado o primeiro plano para desenvolver a primeira rede de computação de pacotes o que chegaria até a internet que usamos nos dias de hoje.

Em meados de 1989, Tim Berners-Lee desenvolveu o WWW world wide web que funciona como um sistema que integra e distribui documentos de hipertextos interconectados. Para se aprofundar mais nesse assunto clique aqui

Clientes e Servidores:

Cliente pode ser definido como um dispositivo ou programa em que há o acesso a internet, por exemplo um usuário pode solicitar a uma página web o aceso a uma página.

Servidor é um dispositivo que fornece serviços para atender as solicitações de clientes.

HTML:

O html é uma linguagem de marcação para páginas na web, ela serve justamente para fornecer a estrutura para diversas páginas na web. Um fato interessante é que muitos ainda confudem o html como uma linguagem de marcação, porém a o html não é capaz de criar estruturas dinâmicas, sendo assim, o html não pode ser considerado como uma linguagem de programação.

Existem 142 tags para serem utilizadas no HTML, dentre as principais temos:

  • A tag < html > é o elemento que define toda a estrutura HTML.
  • A tag < head > contém as informações sobre o título e a codificação de caracteres da página.
  • A tag < body > apresenta todas as informações que estarão sendo apresentadas na tela.

Outras tags populares muito usadas são:

  • Tags de Cabeçalho – elas variam entre < h1 > e < h6 >, em que a escala de h1 a h6 varia no nível de importância das informações a serem apresentadas como título.
  • Tags de parágrafo – são abertas usando a tag < p >< /p>. Também existem as tag < b > < /b > para negrito, < i >< /i > para itálco
  • Tags de lista – possuem diferentes variações. Use a tag < ol > para uma lista ordenada, e a tag < ul > para uma lista sem ordem definida. Então, envolva cada item da lista com a tag < li >
